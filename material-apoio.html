Repositório do GitHub

  • 1 - Faça o download e instale o git bash (https://git-scm.com/downloads)
  • 2 - Abra um terminal, pode ser dentro do Visual Studio Code (terminal -> novo termina)
  • 3 - Faça o clone do repositório (https://github.com/rodrigoh/progII_C)
  • 4 - Verifique se o comando retornou Receiving objects: 100%
  • 5 - Abra a pasta onde realizado o clone (caminho\ProgII_C) no VScode com a opção abrir pasta no menu arquivo.
  • Os arquivos da aula devem estar todos ali dentro no final do processo.
